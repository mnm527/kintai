☁ Supabase 複数端末同期

● 未接続 接続すると複数端末でリアルタイム同期されます
▶ Supabase 初期設定手順

1. supabase.com でプロジェクトを作成

2. SQL Editor で以下を実行：

create table if not exists employees (
  id bigint primary key,
  name text not null
);

create table if not exists records (
  id text primary key,
  employee_id bigint not null,
  employee_name text not null,
  date text not null,
  clock_in text,
  clock_out text,
  break_min integer default 0,
  remarks text default '',
  unique (employee_id, date)
);

create table if not exists settings (
  id integer primary key default 1,
  scheduled_min integer default 480
);

alter table employees enable row level security;
alter table records enable row level security;
alter table settings enable row level security;

create policy "public_access" on employees for all using (true) with check (true);
create policy "public_access" on records for all using (true) with check (true);
create policy "public_access" on settings for all using (true) with check (true);

3. Settings → API から Project URL と anon public キーをコピーして上に入力

4. 「接続する」→ 既存データがある場合は「ローカルデータをアップロード」で同期
